# FutaFilter

https://futaguard.github.io/FutaFilter/filter.txt

- 專為臺灣人設計的廣告阻擋規則
- 本規則專注於補強 AdGuard 基本規則的遺漏

# 使用方式

打開 AdGuard 或是其他支援 AdBlock 句子的 Blocker
